# ☕ Break Music

Поместите сюда MP3-файлы для музыки во время перерыва.

## Как это работает:
- При входе в режим "Перерыв" автоматически включается случайный трек
- Музыка играет в фоне пока ведущий вводит баллы
- При выходе из перерыва музыка останавливается

## Рекомендации:
- Спокойная фоновая музыка
- Формат: MP3
- Громкость будет приглушена автоматически

## Пример файлов:
- chill_1.mp3
- lounge.mp3
- waiting.mp3
- background_1.mp3

Если папка пустая - будут использованы треки из timer-music/
